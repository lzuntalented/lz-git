# 随心码
一款基于 Javascript 开发的 git 代码管理平台
## 开发
```bash
# 服务端 需要先安装mysql，创建名为lzgit的数据库
cd packages/server
# 默认端口3000
npm start

# 前端  端口3001
cd packages/frontend
npm start

# 访问 http://localhost:3001/

```
